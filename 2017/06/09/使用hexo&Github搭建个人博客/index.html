环境搭建

GitHub Windows下载

配置github环境变量，设置ssh公钥请参考此文：windows系统下git安装和配置

安装Node.JS

hexo本质上是一个Node.JS模块，所以需要先安装Node.JS

安装hexo

安装完github后，桌面上会出现Git Shell，打开后输入npm命令就可以安装

部署至Github

1.申请github账号，创建仓库

具体请参考：官方文档

2.将安装github时生成的ssh密钥添加至github仓库

密钥保存在.ssh目录的id_rsa.pub文件中，.ssh目录一般在C:\Users(计算机用户名).ssh目录下。

注意：在添加时要勾选允许写入操作(allow write access)的选项
